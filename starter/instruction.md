# GitHub Copilot Instructions

This project is a Flask-based Sudoku game.

When generating code:

- Follow PEP 8 coding standards.
- Use modular and reusable functions.
- Keep functions short and focused.
- Preserve existing functionality unless explicitly changing it.
- Use descriptive variable and function names.
- Add comments only where the logic is complex.
- Handle errors gracefully using try/except where appropriate.
- Avoid duplicate code.
- Keep HTML, CSS, JavaScript, and Python organized.
- Maintain compatibility with the existing Flask application.
- Prefer readable code over clever code.
- Ensure new features are responsive and accessible.
- Write code that is easy to test.
